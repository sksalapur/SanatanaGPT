1. Number Systems:
   - Binary, Octal, Hexadecimal, 1’s and 2’s Complement

   - Fetch-Decode-Execute Cycle, Pipelining

3. Memory Hierarchy:
   - Cache, Main Memory, Virtual Memory, Associative Mapping

4. Addressing Modes:
   - Immediate, Direct, Indirect, Indexed

5. I/O Organization:
   - Programmed I/O, Interrupts, DMA
